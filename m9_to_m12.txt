
Go to the ZMI (http://.../manage_main):

  1. Remove the object "culture/.catalog"

  2. Add the following "External Method" in the Zope root:

     id      : manage_update_to_m12
     module  : ikaaro.manage_update_to_m12
     function: manage_update_to_m12

  3. Run it (http://.../culture/manage_update_to_m12)

  4. Create a dummy ikaaro instance in the Zope root.

  5. Copy the ".catalog" object from the dummy instance and paste it
     into the culture instance.

  6. Create the following "External Method" in the Zope root:

     id      : manage_ikaaro
     module  : ikaaro.manage_ikaaro
     function: manage_ikaaro

  7. Run it (http://.../culture/manage_ikaaro)

  8. Click on the "Update" button.


Go to the Back-Office:

  9. Authenticate as admin

  10. Update the catalog (http://.../culture/;catalog_form)
